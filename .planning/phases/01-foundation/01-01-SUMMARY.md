## Key Decisions

| Decision | Rationale |
|----------|-----------|
| `tokio flavor = "current_thread"` | Single RS485 bus = sequential polling; eliminates `Send` bounds on serial handles |
| `reqwest features = ["rustls"]` | Feature was renamed from `rustls-tls` in reqwest 0.13.x; avoids OpenSSL for ARM cross-compile |
| `tokio-modbus default-features = false, features = ["rtu"]` | Disables TCP feature; keeps binary lean |
| No `chrono` dependency | `std::time::SystemTime` sufficient for epoch seconds |
